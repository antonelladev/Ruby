# Logo oficial

Coloca aquí el logo oficial de RUBY con el nombre exacto:

```
assets/logo/logo.png
```

Recomendado: PNG con fondo transparente, altura aproximada de 40–56px de alto útil.

Mientras este archivo no exista, el navbar muestra automáticamente el wordmark
de texto "RUBY" como respaldo — el diseño no se rompe. La detección ocurre en
`initLogo()` dentro de `js/interactions.js`.
